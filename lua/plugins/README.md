# 插件

## lazy

打开lazy管理页: `:Lazy`

## treesitter

### 功能

1. 打开`zo` /关闭`zc` 代码块
2. 格式化代码, v模式下快捷键`=`
3. 安装`:TSInstall [language]`, 例如: `:TSInstall c`
4. 手动开启代码高亮: `TSBufToggle highlight`
5. 查看模块信息: `TSModuleInfo`, 查看安装信息: `TSInstallInfo`
